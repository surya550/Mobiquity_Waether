How to use

On Home Page, you have to add a location, with the help of a button("+ symbol")

  • If permission is provided, & location services is enabled map will take you to current location
  • location is added to db based on reverse geocoding option
  • you can search by location name from saved locations

Home Page, you will see a list of locations, which you had saved

  • you can delete the item by long click on the particular row
  • Click the particular row, to fetch, the current weather details
  • Click on the = next 7 days button, you will get next 7 days weather info.

In Setting Page, You can configure the following things

  • Measurement Unit: which will be fed to the api, on detail screen
  • "Clear Data": Once checked, all the items will be deleted from the local db, & again it will be set to false, after deletion
